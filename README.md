## 项目简介

CueMate 是一款开源的智能语音面试助手 - AI 驱动的实时面试训练工具，专注于帮助求职者通过系统化的模拟面试训练和面试准备，提升面试技能和表现。

### 安装步骤

1. 根据芯片类型下载对应的 DMG 安装包
2. 双击打开 DMG 文件，按照安装向导完成安装（约 5 分钟）
3. 使用内置账号登录（默认用户名：`admin`，密码：`cuemate`）
