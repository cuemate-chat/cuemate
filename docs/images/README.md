## 截图制作说明

1. **截图工具**：
   - macOS：使用 Command+Shift+4 截取指定区域
   - 或使用 Command+Shift+5 选择截图区域

2. **图片处理**：
   - 使用标注工具（如 Skitch、Preview）添加箭头、高亮等标注
   - 确保敏感信息（如真实简历内容）已脱敏处理

3. **文件命名**：
   - 保持与文档中引用的文件名一致
   - 使用 PNG 格式以确保清晰度

4. **提交要求**：
   - 图片文件大小建议控制在 500KB 以内
   - 如果图片较大，可以适当压缩（推荐使用 TinyPNG）
